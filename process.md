# End-to-End System Process

1. Electrical device is powered ON
2. Current flows through ACS712 sensor
3. Sensor generates analog voltage
4. ESP32 reads and processes sensor data
5. ESP32 sends data to cloud
6. Cloud stores data by location and time
7. AI module analyzes historical usage
8. Dashboard displays real-time and monthly data
9. Alerts notify users when limits are crossed
